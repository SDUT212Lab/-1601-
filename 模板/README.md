#### 安装教程

1. 函数库模板的使用，可以将KEIL的魔术棒中的STM的芯片换一些和STM的FLASH换一下即可直接使用
2. 函数库中的函数文件夹直接放在模板的USAER文件夹下，将.c文件添加到项目的user文件目录中，在魔术棒的c++选项中添加包含这个文件夹的头文件
